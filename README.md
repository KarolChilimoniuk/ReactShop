### How does it run?
1. Clone this repo.
2. Go to a repo file by console.
3. Install packages by writing command in console:
```
npm install
```
4. If you want to see live view, use command:
```
npm start
```
### Additional comands
To test app
```
npm test 
```

To create build folder for the production
```
npm build
```
